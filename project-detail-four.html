Vancouver Discovery

Graphic Design

Graphic Designer

TOOLS

Members

Project Overview

Vancouver Discovery is an interactive brochure that provides a brief overview of Vancouver, where to go, and what to eat. The objective of this work is to exhibit Adobe InDesign skills in creating an effective layout, adding photos, creating and designing hover buttons, embedding movies, and inserting linkages to internal and external sources.

01 Ideatation

Generate ideas

As an international student, Vancouver has always been a fascinating city for me to explore. That is why I chose Vancouver Discovery as the focus of my work. To begin this project, I spent some time learning about the subjects discussed in this post as well as the major interactive layout functions that would be used in my project.

Information Collections

I design user scenarios to better understand people so that I can build more effective user flows. User Scenarios assist me in mapping out user pain points, motivations, and product development potential.

02 Work
Process

Sketchs

The layout of the assignment will be sketched first. Then I'll be able to see if there are any design faults that need to be fixed before I start using Adobe InDesign. Sketching also allows me to produce ideas for future use. This is a rapid method for graphically illustrating an image, idea, or principle.

Input prototype & Testing

To properly use and exhibit InDesign's interactive capabilities, prototypes such as links and animation must be introduced after putting photos and written material into the pages.

To guarantee that the interactive layout works properly, prototypes are tested. All of the buttons and links work properly and lead to both internal and external pages.

03 Final Product

The finished product is a digital brochure that can be accessed via the Indesign link or PDF file provided, which includes all functional buttons, videos, and hyperlinks. However, the layout only works with the link. Videos could not be played in the pdf version, and buttons navigated to the wrong page. This is a limitation of Adobe Indesign; it allows users to create interactive layouts, but errors appear when exported as PDF files. Overall, the project is still on track to meet its objectives, as optimized Adobe Indesign skills are used to create an interactive layout. The final product in Indesign link works fine.
